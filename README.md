# Insurance Claims Processing System

A microservices-based insurance claims platform built with Java 17 and Spring Boot.

## Services
- auth-service
- customer-service
- policy-service
- claim-service
- document-service
- notification-service
- audit-service
- api-gateway

## Tech Stack
- Java 17
- Spring Boot
- Spring Security
- JWT
- PostgreSQL
- RabbitMQ
- Docker
- Docker Compose
- Prometheus
- Grafana
- Jenkins

## Features
- Authentication and authorization
- Customer and policy management
- Claim creation and status updates
- Document upload
- Event-driven notifications with RabbitMQ
- Audit logging
- API Gateway
- Monitoring with Prometheus and Grafana
